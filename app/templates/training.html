Familiarizing Yourself with the Game

This page will explain how this game works.

The Cards

The cards used in the game are from the game Set and have a series of properties.
  • Number of Shapes: 1, 2, 3
  • Shading: open, striped, solid
  • Type of Shape: diamond, oval, squiggle
  • Color: red, green, purple

Image from: https://geekandsundry.com/the-card-game-that-puzzled-mathematicians-for-decades/

The Game

The game consists of two colored bins (red and orange) that you must sort the cards into based on their properties.

The Demonstration Phase

You have a helper robot who will start by demonstrating where several cards belong. Pay attention here because it will give you clues as to the rules for which card goes where!

The Trial Phase

During the trials, you get to choose where each card goes (left). Your helper robot may give you some feedback after each attempt to help you improve (right).
